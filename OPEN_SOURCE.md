# Open-source model

The reference implementation and protocol are MIT licensed. Anyone may inspect, run, copy, fork, mirror, redistribute, or independently reimplement them.

The Python node is not the authority. Compatibility is defined by the public schemas, canonical encoding, validation rules, and conformance vectors.

GitHub is a convenient first distribution channel, not protocol ownership. Releases should be mirrored, checksummed, reproducibly built, and independently verifiable.

A healthy network requires multiple independent implementations, archive operators, coordinators, interfaces, and maintainers. No trademark, domain, hosted API, founder key, trust store, or organisation is required for base technical participation.
